Foi utilizado o codigo por texto, um arduino uno, uma placa de ensaio, fios, um resistor e um buzzer.
Como a proposta desse projeto era uma criação autoral, decidimos utilizar um buzzer para reproduzir toda a melodia da música do Super Mário, primeiro, utilizamos um violão para tocar
todas as notas da música, uma por uma, enquanto as mesmas foram anotadas, depois, utilizamos uma tabela com a frenquencia de cada uma das notas para poder colocá-las no código,
juntamente com as suas durações e pausas, que exigiu que ouvissemos repetidas vezes a melodia para poder ajustar o mais fiel à música original possível, iremos montar esse projeto 
fisicamente em sala dia 28/11/2025 para apresentar aos alunos e docentes.
